Tree Traversal

Three ways to visit every node in a binary tree, each giving a different order.

All O(n)Space O(h)

Three traversals — three different visit orders: Inorder (Left→Root→Right) gives sorted output for BSTs. Preorder (Root→Left→Right) is useful for copying a tree. Postorder (Left→Right→Root) is used when deleting nodes.

Comparison

Inorder: Left → Root → Right
Result: 1, 3, 4, 6, 7, 8, 10, 13, 14

Preorder: Root → Left → Right
Result: 8, 3, 1, 6, 4, 7, 10, 14, 13

Postorder: Left → Right → Root
Result: 1, 4, 7, 6, 3, 13, 14, 10, 8

function inorder(node) {
  if (!node) return;
  inorder(node.left);   // ←
  visit(node);          // ●
  inorder(node.right);  // →
}
function preorder(node) {
  if (!node) return;
  visit(node);           // ●
  preorder(node.left);   // ←
  preorder(node.right);  // →
}
