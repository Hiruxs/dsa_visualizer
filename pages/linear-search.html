Linear Search

Check each element one by one until found or all checked.

Time O(n)Space O(1)Works on Unsorted
Linear Search — scan from left to right
Speed: Fast
Enter a target value, then press Auto Search or Step.

The simplest search — no preprocessing needed, works on any array (sorted or not). Best case O(1) if the target is first; worst case O(n) if last or absent. Try: 42, 6, or 99.

Pseudocode — highlighted lines follow the animation
FUNCTION linearSearch(arr, target): FOR i FROM 0 TO length(arr) - 1: // check current element IF arr[i] = target THEN RETURN i // found at index i END FOR RETURN -1 // not found END FUNCTION
