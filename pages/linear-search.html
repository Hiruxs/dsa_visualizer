Linear Search

Check each element one by one from the beginning until the target is found.

Time O(n)Space O(1)Best O(1)
Visualization
Speed:Fast
Enter a value and press Search to begin.

Linear Search works on any array — sorted or unsorted. It scans from index 0 to the end, comparing each element with the target. Simple but slow for large datasets. Use Binary Search for sorted arrays to get O(log n) instead.

Code
function linearSearch(arr, target) {
  for (let i = 0; i < arr.length; i++) {
    // Check each element
    if (arr[i] === target) {
      return i; // Found at i
    }
  }
  return -1; // Not found
}
// Time:  O(n)
// Space: O(1)
// Best:  O(1) — found at index 0
// Worst: O(n) — last element or not found
When to use
