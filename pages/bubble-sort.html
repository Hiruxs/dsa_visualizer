Bubble Sort

Time O(n²)Space O(1)
Visualization
Speed: Fast
Press Start to begin sorting, or Step to go one step at a time.

How it works: Compare adjacent pairs. If left > right, swap them. Repeat across the whole array. After each full pass, the largest unsorted element 'bubbles up' to its correct position. Takes O(n²) time — not efficient for large arrays, but very easy to understand.

Code
function bubbleSort(arr) {
  let n = arr.length;
  for (let i=0; i<n-1; i++) {
    for (let j=0; j<n-i-1; j++) {
      // Compare adjacent
      if (arr[j] > arr[j+1]) {
        // Swap them
        [arr[j], arr[j+1]] =
        [arr[j+1], arr[j]];
      }
    }
  }
  return arr;
}
// Time:  O(n²)
// Space: O(1)
