Bubble Sort

Repeatedly compare adjacent pairs and swap if out of order — larger values bubble to the end.

Time O(n²)Space O(1)Stable
Bubble Sort — largest element bubbles to end each pass
Speed: Fast
Ready — use Step to go one step at a time, or Auto to run fully.

Each full pass guarantees the largest unsorted element is placed at the correct position. The optimization if(!swapped) break exits early when the array is already sorted — best case O(n).

Pseudocode — highlighted lines follow the animation
PROCEDURE bubbleSort(arr): n ← length(arr) FOR i FROM 0 TO n-2: swapped ← false FOR j FROM 0 TO n-i-2: // compare adjacent pair IF arr[j] > arr[j+1] THEN swap arr[j] and arr[j+1] swapped ← true // no swaps = already sorted IF swapped = false THEN BREAK END FOR END PROCEDURE
