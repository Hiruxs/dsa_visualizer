Heap Sort

Build a max heap then extract the max repeatedly — sorts in-place.

Time O(n log n)Space O(1)Not Stable
Heap Sort — Phase 1: build heap, Phase 2: extract max
Speed: Fast
Ready — use Step to go one step at a time, or Auto to run fully.

Two phases: (1) Build converts array to max heap in O(n). (2) Extract swaps root (max) to the end, shrinks heap by 1, and heapifies — O(log n) per step × n steps = O(n log n). Watch the green sorted portion grow from right to left!

Pseudocode — highlighted lines follow the animation
HEAPSORT(A) 1 BUILD-MAX-HEAP(A) // Phase 1: O(n) 2 for i = A.length downto 2 // Phase 2 3 exchange A[1] with A[i] // max to end 4 A.heap_size = A.heap_size - 1 5 MAX-HEAPIFY(A, 1) // restore BUILD-MAX-HEAP(A) 1 A.heap_size = A.length 2 for i = A.length/2 downto 1 3 MAX-HEAPIFY(A, i) MAX-HEAPIFY(A, i) 1 l = LEFT(i), r = RIGHT(i) 2 if l <= heap_size and A[l] > A[i] 3 largest = l else largest = i 4 if r <= heap_size and A[r] > A[largest] 5 largest = r 6 if largest != i 7 exchange A[i] with A[largest] 8 MAX-HEAPIFY(A, largest)
