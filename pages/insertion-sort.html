Insertion Sort

Pick one element at a time and insert it into the correct position in the sorted portion.

Time O(n²)Space O(1)StableBest O(n)
Insertion Sort — build sorted portion one element at a time
Speed: Fast
Ready — use Step to go one step at a time, or Auto to run fully.

Like sorting a hand of cards — you pick one card and slide it left until it's in the right spot. The left side is always sorted. Excellent for small arrays or nearly-sorted data (best case O(n)).

Pseudocode — highlighted lines follow the animation
PROCEDURE insertionSort(arr): n ← length(arr) FOR i FROM 1 TO n-1: key ← arr[i] // element to place j ← i - 1 // shift larger elements right WHILE j >= 0 AND arr[j] > key: arr[j+1] ← arr[j] j ← j - 1 // insert key in correct position arr[j+1] ← key END FOR END PROCEDURE
