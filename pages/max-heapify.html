Max Heapify

Fix a single max-heap violation by sifting one node down to its correct position.

Time O(log n)Space O(1)
Max Heapify — sift one node down to correct position
Speed: Fast
Ready — use Step to go one step at a time, or Auto to run fully.

Assumes both subtrees are already valid max heaps. Compares the node with its largest child — if the child is larger, swap and continue recursively. The node sifts down until it's larger than both children or becomes a leaf.

Pseudocode — highlighted lines follow the animation
MAX-HEAPIFY(A, i) 1 l = LEFT(i) // 2i + 1 2 r = RIGHT(i) // 2i + 2 // Find largest among node, left, right 3 if l <= A.heap_size and A[l] > A[i] 4 largest = l 5 else largest = i 6 if r <= A.heap_size and A[r] > A[largest] 7 largest = r // If root not largest: swap, recurse down 8 if largest != i 9 exchange A[i] with A[largest] 10 MAX-HEAPIFY(A, largest)
