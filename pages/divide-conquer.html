Divide and Conquer

Break the problem in half, solve one part, combine — O(log n) per level.

Binary Search O(log n)Powerful Paradigm
Divide & Conquer — Binary Search demo
Speed: Fast
Ready — use Step to go one step at a time, or Auto to run fully.

Divide: Split at midpoint. Conquer: Recursively search one half (not both — unlike Merge Sort). Combine: No combination needed here. Also used in: Merge Sort, Quick Sort, Fast Fourier Transform, and Strassen matrix multiplication.

Pseudocode — highlighted lines follow the animation
// Binary Search — classic D&C BINARY-SEARCH(A, target, lo, hi) 1 if lo > hi 2 return -1 // base case: not found 3 mid = (lo + hi) / 2 // DIVIDE 4 if A[mid] = target // CONQUER 5 return mid // Recurse on ONE half only 6 if A[mid] > target 7 return BINARY-SEARCH(A, target, lo, mid-1) 8 else 9 return BINARY-SEARCH(A, target, mid+1, hi) // T(n) = T(n/2) + O(1) → O(log n)
