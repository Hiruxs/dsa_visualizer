Array

A collection of elements stored in contiguous memory locations, accessible via an index.

Access O(1) Search O(n) Insert O(n)
Visualization
Array initialized with 6 elements. Click any operation above.

What is an Array? An array stores elements in a fixed block of memory. Each element sits at a specific index starting from 0. Because the memory is contiguous, you can jump to any element instantly using its index — that's why access is O(1). However, inserting or deleting requires shifting elements, making it O(n).

How it works in memory
// Array in memory
Index:  [0] [1] [2] [3]
Value:  [5] [3] [8] [1]
Addr: 100 104 108 112

// Access element at index i
address = base + (i × size)
// 100 + (2 × 4) = 108 → value 8

// JavaScript
let arr = [5, 3, 8, 1];
arr[2];      // → 8, O(1)
arr.push(9); // → end, O(1)
arr.splice(1,0,7); // insert, O(n)
Operations
1
Access: Directly jump using index formula. O(1)
2
Insert at end: Place after last element. O(1)
3
Insert at index: Shift all elements right first. O(n)
4
Delete: Remove and shift elements left. O(n)
