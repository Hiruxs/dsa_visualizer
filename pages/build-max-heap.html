Build Max Heap

Convert any array into a max heap in O(n) using bottom-up heapify.

Time O(n)Space O(1)In-Place
Build Max Heap — enter any array
Speed: Fast

Start heapify at the last non-leaf node (n/2-1) and work backward to index 0. Leaf nodes are already valid heaps. This bottom-up approach is O(n) — better than inserting n elements one by one, which is O(n log n).

Pseudocode — highlighted lines follow the animation
BUILD-MAX-HEAP(A) 1 A.heap_size = A.length 2 for i = A.length/2 downto 1 3 MAX-HEAPIFY(A, i) MAX-HEAPIFY(A, i) 1 l = LEFT(i) // 2i + 1 2 r = RIGHT(i) // 2i + 2 3 if l <= A.heap_size and A[l] > A[i] 4 largest = l 5 else largest = i 6 if r <= A.heap_size and A[r] > A[largest] 7 largest = r 8 if largest != i 9 exchange A[i] with A[largest] 10 MAX-HEAPIFY(A, largest)
