Delete & Search
Singly linked list initialized with 4 nodes.

Singly Linked List: Each node has data and a single next pointer. Memory-efficient (one pointer per node), but you can only traverse one direction. To go back, you must restart from HEAD.

Singly vs Doubly

Singly: 1 pointer per node (next). Less memory. One-direction traversal.

Doubly: 2 pointers per node (prev + next). More memory. Both directions traversal, easier deletion.
