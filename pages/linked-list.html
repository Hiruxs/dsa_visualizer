Linked List

A chain of nodes where each node holds data and a pointer to the next node.

Insert Head O(1)Search O(n)Insert Middle O(n)
Visualization
HEAD
Linked list with 4 nodes. HEAD → 10 → 20 → 30 → 40 → NULL

Key difference from arrays: Nodes are not stored in contiguous memory. Each node stores a pointer to the next. This means insertion/deletion at head is O(1) — no shifting needed. But accessing the Nth element requires traversing from HEAD.

Node Structure
class Node {
  constructor(val) {
    this.data = val;
    this.next = null;
  }
}

class LinkedList {
  constructor() {
    this.head = null;
  }

  prepend(val) {
    let node = new Node(val);
    node.next = this.head;
    this.head = node;
    // O(1)
  }

  append(val) {
    // Traverse to end O(n)
    let node = new Node(val);
    if (!this.head) {
      this.head = node; return;
    }
    let cur = this.head;
    while (cur.next) cur = cur.next;
    cur.next = node;
  }
}
