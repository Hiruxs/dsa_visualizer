Stack

A Last In, First Out (LIFO) data structure — like a stack of plates.

Push O(1)Pop O(1)Peek O(1)
Visualization — Stack grows upward
← TOP (last in, first out)
Bottom of Stack
Stack has 3 elements. TOP is 30.

Real-world uses: Browser back button (each page is pushed; back pops), undo operations in editors, function call stack in programming, and expression evaluation.

Code
class Stack {
  constructor() {
    this.items = [];
  }

  push(val) {
    this.items.push(val);
    // O(1)
  }

  pop() {
    if (this.isEmpty()) return null;
    return this.items.pop();
    // O(1)
  }

  peek() {
    return this.items[
      this.items.length - 1
    ]; // O(1)
  }

  isEmpty() {
    return this.items.length === 0;
  }
}
Key Concept
↑
Push: Add to the TOP of the stack
↓
Pop: Remove from the TOP only
👁
Peek: See the TOP without removing it
