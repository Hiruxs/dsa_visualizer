Stack

Last In, First Out — push to top, pop from top.

Push O(1)Pop O(1)Peek O(1)
Stack — Last In First Out (LIFO)
Speed: Fast
Ready — use Step to go one step at a time, or Auto to run fully.

Think of a stack of plates — you always add and remove from the top. Used for undo systems, function call stacks, expression evaluation, and depth-first search.

Pseudocode — highlighted lines follow the animation
STACK: arr[], top = -1 PROCEDURE push(val): top ← top + 1 arr[top] ← val END PROCEDURE FUNCTION pop(): IF top = -1 THEN raise error val ← arr[top] top ← top - 1 RETURN val END FUNCTION FUNCTION peek(): IF top = -1 THEN raise error RETURN arr[top] END FUNCTION FUNCTION isEmpty(): RETURN top = -1 END FUNCTION
