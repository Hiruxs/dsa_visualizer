Every recursive function needs: (1) a base case (when to stop) and (2) a recursive case (how to reduce the problem). Each call is pushed onto the call stack. When the base case is reached, calls unwind back — returning and combining results.

Pseudocode — highlighted lines follow the animation
// Factorial: n! = n × (n-1)! FACTORIAL(n) 1 if n <= 1 2 return 1 // base case 3 return n * FACTORIAL(n - 1) // Fibonacci: F(n) = F(n-1) + F(n-2) FIB(n) 1 if n <= 1 2 return n // base case 3 return FIB(n-1) + FIB(n-2) // Sum: SUM(n) = n + SUM(n-1) SUM(n) 1 if n <= 0 2 return 0 // base case 3 return n + SUM(n - 1)
