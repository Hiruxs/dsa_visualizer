Merge Sort

Divide in half recursively, then merge sorted halves together.

Time O(n log n)Space O(n)Stable
Merge Sort — divide then merge sorted halves
Speed: Fast
Ready — use Step to go one step at a time, or Auto to run fully.

Divide: Split at midpoint until 1-element arrays (trivially sorted). Merge: Combine two sorted halves by picking the smaller front element each time. Guaranteed O(n log n) on all inputs — unlike Quick Sort which can degrade to O(n²).

Pseudocode — highlighted lines follow the animation
MERGESORT(A, p, r) 1 if p < r 2 q = (p + r) / 2 3 MERGESORT(A, p, q) 4 MERGESORT(A, q+1, r) 5 MERGE(A, p, q, r) MERGE(A, p, q, r) 1 n1 = q - p + 1 2 n2 = r - q 3 create arrays L[1..n1] and R[1..n2] 4 for i = 1 to n1 5 L[i] = A[p + i - 1] 6 for j = 1 to n2 7 R[j] = A[q + j] 8 i = 1, j = 1, k = p 9 while i <= n1 and j <= n2 10 if L[i] <= R[j] 11 A[k] = L[i]; i = i + 1 // pick left 12 else 13 A[k] = R[j]; j = j + 1 // pick right 14 k = k + 1 15 copy remaining L or R into A
