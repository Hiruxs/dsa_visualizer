Quick Sort

Avg O(n log n)Worst O(n²)Space O(log n)
Visualization
Speed: Fast
Press Start to begin sorting, or Step to go one step at a time.

How it works: Pick a pivot element. Rearrange the array so all elements smaller than the pivot go left, and all larger go right. Then recursively sort the left and right halves. Average case is O(n log n) — very fast in practice.

Code
function quickSort(arr, lo, hi) {
  if (lo >= hi) return;
  let pivot = arr[hi];
  let i = lo - 1;
  for (let j=lo; j<hi; j++) {
    if (arr[j] <= pivot) {
      i++;
      [arr[i],arr[j]] = [arr[j],arr[i]];
    }
  }
  // Place pivot in final position
  [arr[i+1],arr[hi]] = [arr[hi],arr[i+1]];
  let p = i + 1;
  quickSort(arr, lo, p-1);
  quickSort(arr, p+1, hi);
}
// Time:  O(n log n) avg
// Space: O(log n)
