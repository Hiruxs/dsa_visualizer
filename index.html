How data is organized and stored in memory

Array

Store elements in contiguous memory locations with index-based access.

→
⊟

Stack

Last In, First Out (LIFO) — like a pile of plates.

→
⊡

Queue

First In, First Out (FIFO) — like a line at a shop.

→
↺

Circular Queue

A queue that wraps around — reuses empty slots efficiently.

→
⬡

Linked List

Nodes connected by pointers — dynamic size, no contiguous memory needed.

→
△

Binary Tree

Each node has at most two children — left and right.

→
⊛

Binary Search Tree

Left < Root < Right — enables fast search in O(log n).

→
▽

Complete Binary Tree

All levels filled except possibly the last, filled left to right.

→
◈

Full Binary Tree

Every node has either 0 or 2 children — never just one.

→
⬠

Heap

A complete binary tree satisfying the heap property.

→
▲

Max Heap

Parent is always greater than or equal to its children.

→
▼

Min Heap

Parent is always less than or equal to its children.

Part 2

Algorithms

Step-by-step procedures for solving problems

↑

Insertion Sort

Build sorted array one element at a time by inserting into correct position.

→
⬆

Bubble Sort

Repeatedly swap adjacent elements until the array is sorted.

→
⚡

Quick Sort

Divide and conquer using a pivot to partition the array.

→
⊕

Merge Sort

Split, sort recursively, then merge — always O(n log n).

→
⊗

Heap Sort

Sort by building a max heap and extracting the maximum repeatedly.

→
⌕

Linear Search

Check each element one by one until the target is found.

→
⤵

Tree Traversal

Visit every node in a tree in a specific order.

→
L→R

Inorder Traversal

Left → Root → Right — gives sorted output for a BST.

→
R→

Preorder Traversal

Root → Left → Right — useful for copying a tree.

→
→R

Postorder Traversal

Left → Right → Root — useful for deleting a tree.

→
∞

Recursion

A function that calls itself to solve smaller subproblems.

→
÷

Divide & Conquer

Break the problem into smaller parts, solve each, then combine.

→
⊞

Build Max Heap

Convert an unsorted array into a valid max heap in O(n).

→
⊡

Max Heapify

Fix a single violation of the max heap property by sifting down.
